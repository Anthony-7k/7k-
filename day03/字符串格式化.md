| 问题                    | 答案                                                                    |
| --------------------- | --------------------------------------------------------------------- |
| **为什么推荐 f-string？**   | 语法简洁、可读性高、性能较好，支持直接嵌入变量和表达式，是 Python 3.6+ 官方推荐的格式化方式。                 |
| **`input()` 返回什么类型？** | **始终返回 `str`（字符串）类型**，即使输入的是数字，也需要使用 `int()` 或 `float()` 转换后才能进行数值运算。 |
